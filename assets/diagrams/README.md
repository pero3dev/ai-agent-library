# assets/diagrams — 図版のエクスポート画像

図の **正本は各ドキュメント本文に埋め込んだ Mermaid コードブロック** です(二重管理を避けるため、ソースファイルはここに置きません)。

このディレクトリには、画像出力が必要になった場合(スライド転用、Mermaid で表現できない図など)の **エクスポート画像のみ** を置きます。

- 命名: `<セクション番号>-<トピック>-<内容>.<拡張子>`(例: `01-agent-loop-overview.svg`)
- 画像を置いた場合、元になった Mermaid ソースのあるドキュメントを特定できるよう、参照元ドキュメントから相対リンクで参照する
- docs 配下に画像ファイルを直接置かない
