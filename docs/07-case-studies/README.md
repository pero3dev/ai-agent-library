# 07-case-studies — ケーススタディ・アンチパターン詳解

具体事例から学ぶセクションです。成功・失敗事例を構造化して分析し、他セクションの一般論を実例で裏付けます。

- **置くもの**: 事例の構造化された分析(背景 → 設計 → 結果 → 教訓)、横断的なアンチパターン集
- **置かないもの**: 一般論・パターン解説(→ 該当セクションへリンク)
