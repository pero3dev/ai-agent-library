# 15-human-ai — 人と AI の協働(認知・個人)

AI を作る・運用する・組織に入れる側ではなく、**AI と日常的に協働する個人の認知と技能**を扱うセクションです。人がなぜ AI の出力を検証せずに受け入れてしまうのか、任せ続けると技能に何が起きるのか、AI 時代に個人はどう価値を出すのか、そして人が人に AI リテラシーをどう教えるのか — 仕組みの手前にある「使う人間の側」を体系化します。

- **置くもの**: オートメーションバイアスとスキル退化(deskilling)の構造と対策、AI 出力を日常的に検証する個人の技法・習慣、AI 時代のエンジニアのキャリア戦略、非エンジニアを含む AI リテラシー研修の設計。全読者に横断する「人間側の基礎」
- **置かないもの**: 承認・レビューの**仕組み**(→ [Human-in-the-Loop 設計](../02-architecture/human-in-the-loop.md)・[チーム導入とレビュー体制](../08-coding-agents/coding-agent-team-adoption.md))、組織のスキル戦略・ポリシー(→ [09-business](../09-business/README.md) のスキルシフト・利用ポリシー)、認知科学の学術詳解、雇用・労働市場の予測(断定的予測はしない)
