# 09-business — ビジネス実務(案件推進)

AI Agent の案件を企画から本番まで推進するための方法論を扱うセクションです。技術設計の前にある「何をやるか」と、後にある「どう本番に届け、投資を判断するか」を体系化します。

- **置くもの**: ユースケース選定、要件定義、PoC → 本番の進め方、ROI とビジネスケースといった案件推進の方法論
- **置かないもの**: 技術的な設計判断(→ [02-architecture](../02-architecture/README.md))、稼働後の運用実務(→ [05-operations](../05-operations/README.md))、個別事例の分析(→ [07-case-studies](../07-case-studies/README.md))
