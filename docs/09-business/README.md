# 09-business — ビジネス実務(案件推進)

AI Agent の**案件と組織**を推進するための方法論を扱うセクションです。技術設計の前にある「何をやるか」と、後にある「どう本番に届け、投資を判断するか」に加え、AI を組織に定着させる側の実務(開発プロセス・チーム構造・資産共有・利用ポリシー・変化管理)も体系化します。

- **置くもの**: ユースケース選定・要件定義・PoC → 本番・ROI といった案件推進の方法論、価格設計、および組織・プロセス・人材(開発プロセス再設計・チームトポロジー・資産共有・利用ポリシー・スキルシフト)
- **置かないもの**: 技術的な設計判断(→ [02-architecture](../02-architecture/README.md))、稼働後の運用実務(→ [05-operations](../05-operations/README.md))、個別事例の分析(→ [07-case-studies](../07-case-studies/README.md))、コーディングエージェントのツール導入(→ [08-coding-agents](../08-coding-agents/README.md))
